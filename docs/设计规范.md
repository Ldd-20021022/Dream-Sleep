## 1. 设计理念

**关键词**: 宁静、温暖、科技感、信任感

**设计原则**:
- 🌙 **夜间优先**: 全平台默认深色模式，减少睡前使用时的蓝光刺激
- 🧘 **减少焦虑**: 柔和的渐变与圆角，避免尖锐的视觉元素
- ✨ **微妙愉悦**: 微交互、粒子效果、渐变——惊喜但不打扰
- 📊 **数据可视化**: 睡眠数据用直观的温度色表达，降低理解成本
- 🎮 **游戏化隐喻**: 守护者、花园、梦境世界——让改善睡眠变得有趣

**情绪板关键词**: Night Sky · Aurora · Deep Ocean · Warm Campfire · Crystal Cave

## 3. 字体系统

### 3.1 字体族

```css
font-family:
  -apple-system,          /* iOS/macOS 系统字体 */
  BlinkMacSystemFont,     /* macOS Chrome */
  'PingFang SC',          /* 苹方 (中文首选) */
  'Microsoft YaHei',      /* 微软雅黑 (Windows fallback) */
  sans-serif;
```

### 3.2 字号层级

| Token | 大小 | 行高 | 用途 |
|-------|------|------|------|
| `text-xs` | 10px | 1.4 | 图表日期、Tab标签 |
| `text-sm` | 12px | 1.5 | 辅助文字、Tag、时间 |
| `text-base` | 14px | 1.6 | 正文、菜单项、输入框 |
| `text-md` | 16px | 1.6 | 卡片标题 |
| `text-lg` | 18px | 1.5 | 页面标题(AppBar) |
| `text-xl` | 22px | 1.4 | 大标题(Logo) |
| `text-2xl` | 26-28px | 1.3 | 数值展示(评分) |
| `text-3xl` | 32px+ | 1.2 | 关键数据(统计卡片) |

### 3.3 字重

| Token | 值 | 用途 |
|-------|-----|------|
| `font-normal` | 400 | 正文、辅助文字 |
| `font-medium` | 500 | 按钮、菜单项 |
| `font-semibold` | 600 | 标题、卡片标题 |
| `font-bold` | 700 | 数值、大标题、评分 |

### 3.4 排版规则

- 卡片标题: 15-16px / semibold / mb: 12px
- 正文行高: 1.5-1.6
- 长文本段落间距: 8-12px
- 数值大字号展示: 26-32px / bold / 中心对齐
- 中文与数字混排: 数字前后留半角空格(视觉优化)
