## 1. 产品概述

**产品定位**: AI 驱动的 CBT-I 睡眠改善平台，面向有睡眠困扰的中国年轻职场人群。

**核心价值**: 用科学方法（认知行为疗法）+ AI 个性化指导 + 游戏化激励，帮助用户建立健康的睡眠习惯。

**目标指标**:

| 指标 | 目标值 |
|------|--------|
| 注册→健康档案完成率 | ≥ 70% |
| 次日留存 | ≥ 50% |
| 7日留存 | ≥ 30% |
| 30日留存 | ≥ 20% |
| 用户平均睡眠评分提升 | +10 分 (30天内) |
| DAU 记录睡眠比例 | ≥ 40% |
| 付费转化率 | ≥ 5% |

## 4. 核心流程

### 4.1 睡眠记录流程

```
[首页] → [点击"记录睡眠"]
  → 输入入睡时间 (默认上一晚23:00)
  → 输入起床时间 (默认今日07:00)
  → 拖选睡眠质量 (1-5)
  → 选择标签 (深睡/易入睡/多梦/夜醒/早醒/精力充沛)
  → 输入备注 (可选)
  → [保存]
  → 后端计算: 时长 + 评分 + AI反馈
  → 返回首页: 显示更新后的评分 + AI洞察
  → 花园浇水 + 守护者属性提升
```

### 4.2 AI 对话流程

```
[首页快捷入口] → [AI教练]
  → 展示建议问题 (快速提问)
  → 用户输入问题
  → POST /api/v1/chat/send
  → 后端: RAG检索相关知识 → 构建system prompt → DeepSeek生成回复
  → 展示AI回复
  → 继续对话 / 新建会话 / 查看历史
```

### 4.3 改善计划参与流程

```
[首页AI洞察] 或 [知识库推荐]
  → 查看推荐计划 → 了解计划详情 (周目标/日任务)
  → [参与计划]
  → 每日: 首页展示今日计划任务 → [打卡] → +5分
  → 完成所有任务 → 计划完成徽章
  → 查看完成历史
```

### 4.4 付费转化流程

```
[我的] → [VIP会员]
  → 对比 Free/Pro/Premium 权益
  → [选择方案]
  → [选择周期] (月付/季付/年付)
  → [支付] → 微信支付
  → 支付成功 → 会员即时生效
  → 解锁高级功能
```

### 4.5 社区互动流程

```
[社区] (通过功能菜单进入)
  → 浏览帖子 (推荐/关注/热门)
  → 搜索话题或用户
  → 查看帖子详情
  → [点赞] / [收藏] / [评论]
  → [发帖]: 选择话题 → 输入内容+睡眠数据 → 发布
  → [关注]用户 → 关注动态流
```

## 5. 功能验收标准

### 5.1 用户认证

| 编号 | 功能 | 验收标准 | 状态 |
|------|------|----------|------|
| AUTH-01 | 用户名密码注册 | 用户名≥2字符，密码≥6位含字母+数字，邮箱格式正确 | ✅ |
| AUTH-02 | 用户名密码登录 | 正确返回JWT双Token，错误5次/15分钟锁定 | ✅ |
| AUTH-03 | 微信一键登录 | code换openid，新用户自动创建，返回is_new_user标志 | ⚠️ 未配密钥时任何code可登录 |
| AUTH-04 | Token刷新 | Refresh Token有效期内换取新Access Token | ✅ |
| AUTH-05 | Token黑名单 | 登出后Access Token加入黑名单，不可复用 | ✅ |
| AUTH-06 | 密码重置 | 邮箱→Token→新密码，1小时有效期 | ⚠️ dev模式直接返回token，无SMTP |
| AUTH-07 | 密码强度验证 | 返回valid/errors/strength四要素 | ✅ |
