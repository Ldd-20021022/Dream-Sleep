## 紧急回滚条件

出现以下任一情况，立即回滚：

- [ ] 5xx 错误率 > 5%
- [ ] 用户无法登录/注册
- [ ] 数据库连接失败
- [ ] AI 服务全部不可用且 fallback 失败
- [ ] 安全漏洞 (如 Token 泄露、XSS 攻击)
- [ ] 微信支付异常

**回滚步骤:**
```bash
# 1. 切换到上一个稳定版本
git checkout <上一个tag>

# 2. 重新构建并部署
docker compose -f docker-compose.prod.yml down
docker compose -f docker-compose.prod.yml up -d --build

# 3. 验证
curl https://your-domain.com/health

# 4. 通知团队
# 在团队群发送回滚通知
```
